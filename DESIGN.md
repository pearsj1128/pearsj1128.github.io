# Design Theme

## Overview
This website uses a clean modern portfolio style.

## Colors
- Main background: light blue-gray
- Main text: dark gray
- Accent color: blue
- Card background: white

## Layout
- Simple header navigation
- Large hero section
- Profile visual section
- About Me section
- Projects cards
- Contact section

## Style
- Rounded corners
- Soft shadows
- Clear spacing
- Responsive layout for mobile screens

## Goal
The page should look like a simple, clean, student portfolio website hosted on GitHub Pages.
