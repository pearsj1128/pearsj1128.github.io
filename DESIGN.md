# Design Theme: Clean Modern Student Portfolio

## Goal
Create a polished but simple personal portfolio website for GitHub Pages. The site should look complete, readable, and professional without requiring any backend, framework, paid API, or external assets.

## Visual Style
- Clean modern portfolio layout
- Light blue-gray page background
- White rounded content cards
- Blue to purple gradient accent
- Soft shadows and generous spacing
- High contrast text for readability
- Sticky top navigation
- Large hero section with profile avatar/card

## Required Sections
- About Me
- Projects & Activities
- CV / Resume
- Courses I Plan to Take
- AI Usage
- Contact

## Accessibility and Responsiveness
- Use semantic HTML sections
- Use a viewport meta tag
- Include meaningful alt text for profile visuals
- Ensure keyboard focus is visible
- Make the layout responsive for mobile screens

## GitHub Pages Constraints
- Static HTML/CSS only
- No backend
- No paid external API
- Keep files in the repository root and assets folder
